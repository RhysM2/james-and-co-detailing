## Overview

A modern, responsive website built for a professional car detailing business based in Llanelli, Wales.

- React 18 + TypeScript
- Vite
- TailwindCSS v4
- Framer Motion
- React Router
- Web3Forms
- Azure Static Web Apps

- Responsive mobile-first design
- Service packages with pricing
- Before/after gallery slider
- Contact form with spam protection
- SEO optimized with sitemap
