## ğŸŒ Deployment

This site is deployed on Azure Static Web Apps with automatic CI/CD through GitHub Actions.

**Deployment Process:**
1. Push to `main` branch triggers GitHub Actions workflow
2. Vite builds the production bundle
3. Azure Static Web Apps deploys automatically
4. Site goes live at https://www.jamescodetailing.com

**Custom Domain Setup:**
- Domain configured through Azure Static Web Apps
- SSL certificate auto-generated by Azure
- DNS CNAME record points to Azure endpoint
